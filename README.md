# Бюджет — личный планер доходов и расходов

Локальное веб-приложение без сервера и зависимостей: один файл `index.html`.
Горизонт 12 месяцев, план/факт по доходам и расходам, бонусный пул и крупные цели.

## Запуск

Открыть `index.html` в Chrome / Edge / Arc (двойной клик). Всё.

## Данные

- Данные живут в `data.json` рядом с приложением (файл **не** коммитится — см. `.gitignore`).
- Первый запуск: вкладка «Настройки» → «Подключить data.json» → выбрать файл. Дальше приложение сохраняет в него автоматически.
- Без подключённого файла данные держатся в localStorage браузера; есть экспорт/импорт JSON.

## Механика

- **Год** — 12 месяцев: доход/расход план и факт, дельта, накопление нарастающим итогом. Клик по строке — детализация месяца.
- **Месяц** — транши доходов (₽ и $), категории расходов, бонусы. Для $-строк кнопка «ЦБ» подставляет курс ЦБ РФ на дату транша (API cbr-xml-daily.ru).
- **Цели и пул** — бонусы копятся в пул; по каждой цели прогноз достижимости при консервативном и среднем сценарии бонусов.
- **Настройки** — плановый курс $, сценарии бонусов, шаблон месяца (повторяющиеся доходы/расходы).

Шаблон применяется к месяцам без фактов; заполненные месяцы не перезаписываются.
